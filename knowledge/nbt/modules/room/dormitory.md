# Room module — Bedrooms (shared)

> Loads **only when the user assigns a shared Bedrooms room to a floor**. Guidance-only:
> build the furnishing yourself with ordinary ops inside the floor's interior (the
> "[Room plan]" lines give the floor's y range). Share the storey with the other assigned
> room if there are two.
>
> **Scale to the floor:** match the bed count + furniture to the room's size — the "[Room
> plan]" line names the space tier (snug / standard / grand) and a furnishing **preset** to
> base the layout on; build that preset in the house decoration's materials. A grand shared
> bedroom is two facing rows down a zoned aisle, not two beds adrift (see
> `14-furnishing-by-space.md`).

A shared bedroom holds **several beds** in one room — an inn floor, barracks, a children's
room, or servants' quarters. The defining move is repetition with a little separation, so
it reads as multiple sleeping spots, not a furniture pile.

**Bed layout — pick to fit the room:**
- A **single row** of beds along one long wall, evenly spaced (a 1-cell gap between beds),
  OR **two facing rows** down the long walls with a central aisle.
- Give each bed a little **separation**: a low partition (a 1–2 high wall stub, a `fence`
  line, a banner divider) or just a per-bed rug so they don't merge.

**Per-bed kit (kept small, repeated):**
- A **nightstand** (a `barrel` or block) with a `lantern`/`candle`.
- A small **rug** (carpet) and a wall **hook/peg** (item frame, a `*_wall` torch holder).

**Shared elements:**
- A **common wardrobe wall**: a run of stacked `barrel`s/`chest`s and framed doors.
- **Windows** spaced down the long side (curtains via banners/wool); below grade use
  `iron_bars`.
- A small **washing** corner (a `cauldron` + barrel) for an inn/barracks feel.

Light each sleeping spot (a lantern per bed, or evenly spaced hanging lanterns) so no bunk
is in the dark. Leave the central aisle clear to walk.
